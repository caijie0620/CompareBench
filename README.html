Overview

CompareBench asks models to choose one answer from A-D by comparing objects, scenes, marked points, or temporal evidence. It is built with two supporting resources: TallyBench for object counting and OmniCaps for date-keyed image captions.

Current official CompareBench splits are CompareTallyBench, CompareGeometryBench, CompareSpatialBench, and CompareTemporalBench.

CompareBench Splits

Split Samples Format Task
CompareTallyBench 600 2x2 grid Choose the image with the most or fewest target objects.
CompareGeometryBench 200 Single image with A-D labels Compare length, width, height, thickness, or diameter.
CompareSpatialBench 100 Single image with A-D labels Compare depth or vertical height above ground.
CompareTemporalBench 300 2x2 grid Choose the earlier or later image across history, landmarks, and public figures.

Supporting Resources

Resource Size Role
TallyBench 2,000 images Standalone object counting benchmark and source for quantity comparison.
OmniCaps 716 images Date-keyed captions and tags for temporal comparison.
HistoryCaps 516 images Historical events and scenes from 1707-2025.
LandmarkCaps 100 images World landmarks ordered by construction or opening dates.
CelebrityCaps 100 images Public figures ordered by birth dates.
